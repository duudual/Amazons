- AlphaBeta
问题：获得所有的move，再计算他们的得分，再选择前k个，我觉得这种方式也是影响效率的极为重要的原因。
解决：
第一阶段（Queen Move）：先只生成皇后的移动位置。对“皇后移动后的位置”进行快速评分，只保留前 $K_1$ 个最好的移动位置。
    好的移动的评分：
    1.灵活性，周围可move的个数
    2.中心性
    3.最好离其他的障碍点与边界点远一些，可以统计3x3范围内的空白点的个数，越多越好
第二阶段（Arrow Shot）：只对这保留下来的 $K_1$ 个位置生成射箭动作。对射箭结果进行评分，保留前 $K_2$ 个最好的完整走法。
